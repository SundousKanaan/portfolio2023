What is this project?

The project was to prototype a web app in correct HTML and CSS, containing the most important information of the ISGA (International Student Guide Amsterdam), so that the students can use the guide faster, easier and always.

Research

To continue the work, we must first make a research about the book, its content, its style, its layout and interview international students, then collect the results.

Improve

I'm happy with my all pages and how they work the way I want with no issues with clear images and the design I want. I could design my all pages according to my idea and improve according to the feedbacks, then I had my final version, which I am proud of. It was nice to see how my work improves through the project.

The code

I was satisfied with my code because it was the first time to create many screens with one css file with different styles. It was a lot of work, but I also had good results, as a prototype.
Note: Size iPhone 6/7/8.

The process

It was a long project with a lot of work, new knowledge and experiences, but in the end I had a nice result that can work fine.
